A Chrome extension. One input. Four uncomfortable numbers. A red exclamation point that appears when you navigate to a shopping site — as if your browser is disappointed in you.

Manifest v3 No network No tracking

How to install

  1. Open chrome://extensions in Chrome.
  2. Toggle Developer mode (top-right).
  3. Click Load unpacked.
  4. Select the extension/ folder from this project.
  5. Pin the icon from the puzzle-piece menu in your toolbar.
  6. Click the icon. Type a number. Suffer.

Shopping-site detection

The icon shows a red ! when the active tab is on one of: amazon, zara, asos, nike, uniqlo, h&m, ebay, etsy, target, walmart, bestbuy, nordstrom, farfetch, ssense, revolve, aritzia, sephora, shopify-hosted stores, and shop.* subdomains. Open the popup on one of those tabs and a small inverted strip names the site back at you.
